PhyNetPy

Coming Soon

PhyNetPy is a Python library that is currently in development. A first release is targeted for early 2023. This library is designed to be an improvement on PhyloNet, as it will include much of the commonly used inference tools first implemented in PhyloNet. New emphasis will be placed on algorithmic performance and ease of use, while still maintaining PhyloNet's "network first" philosophy. Since PhyNetPy will be released on PyPi, installing will be simply using pip on the command line like so: 

            > pip install PhyNetPy
