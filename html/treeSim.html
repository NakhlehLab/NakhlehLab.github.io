In PhyloGenPy, the module for simulating trees (conditioned on either time or extant taxa) is the BirthDeath module. This module contains multiple models under which trees may be generated, the most basic of which is the constant rate birth death sampling approach (the class CBDP).


To sample any amount of trees under CBDP, we need a birth rate, extinction rate, a number of taxa, and optionally a sampling rate. All rates should be a number between 0 and 1. A normal usage is shown below: 

                        from phylogenpy.BirthDeath import CBDP
        
                        # a birth rate of .05, an extinction rate of .01, 
                        # 6 extant taxa, and the default 100% sampling rate
                        model = CBDP(.05, .01, 6) 
                        
                        model.generateTree() #to see the network, call .printGraph() on the result
                        model.generateTrees(100) #to generate 100 trees
        
                        #Use of the optional sampling rate parameter
                        model2 = CBDP(.05, .01, 6, .5) 
        
                        #to generate trees under the Yule model, set extinction rate to 0
                        yuleModel = CBDP(.05, 0, 6)
